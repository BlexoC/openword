# OpenWord SPA

A small single page application that fetches word definitions from a public API and renders them dynamically.

## Features

- Search for a word using a form submission
- Fetch API requests with `async/await`
- Error handling with `try/catch`
- Interactive button event listeners
- DOM updates with modern JavaScript

## Run

Open `index.html` in your browser. The application works as a static SPA without a build step.
